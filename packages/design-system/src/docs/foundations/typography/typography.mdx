# 🎬 Introdução

A maior parte dos elementos utilizados em sites e aplicativos **são compostas primariamente de textos**,
portanto deve ser selecionada com cuidado, pois pode impactar diretamente a experiência do usuário em contato com a aplicação.

Os padrões utilizados nas nossa interfaces são divididos em duas categorias, sendo elas:

-   Headline
-   Text

Veja o documento a seguir para ver mais detalhes.
