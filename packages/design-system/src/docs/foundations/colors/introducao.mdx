# 🎬 Introdução

Para manter a **consistência entre interfaces e componentes** em projestos digitais, existem um padrão de cores, descrito aqui, que deve ser utilizado.
Dessa forma evitamos o uso de muitas cores diferentes ou até cores com o mesmo significado com tons diferentes, como, por exemplo, uma mensagem de alerta.

O ponto principal dessa definição, é **garantir a consistência visual** dessas interações.
Assim conseguimos **diminuir a carga cognitiva (tempo de aprendizado) da usuário** durante o uso dos nossos serviços,
sem que haja comprometimento da experiência.
