# ➗ WidgetContainer ➗

---

### Para que serve
O `WidgetContainer` é um componente versátil que permite criar contêineres estilizados e organizados por diferentes funcionalidades.
Ele pode ser utilizado para agrupar elementos visuais e componentes dentro de uma seção bem definida, com opções de personalização como cor de fundo.
